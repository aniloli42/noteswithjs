# Note with Js

Note with Js is the website where we can quickly add notes. This website is made with pure HTML, CSS and Javascript.

## Technologies

- HTML
- CSS
- JS

## Features

- Quick Create Note
- Edit Note
- Delete Note
- Note Searching
